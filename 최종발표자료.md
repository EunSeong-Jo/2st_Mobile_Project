## 📱 프로젝트 개요

### 프로젝트명
**알바대 (Alba-Dae)** - 동양미래대학교 학생 특화 아르바이트 매칭 플랫폼

### 개발 기간
2025.10.05 ~ 2025.12.03 (2개월)

### 개발 팀

- 개발 환경: Android Studio, Java
- 데이터베이스: SQLite

### 프로젝트 목표
동양미래대학교 재학생과 주변 고용주를 연결하는 **학교 특화형 아르바이트 매칭 앱** 개발

## 🎯 핵심 기능

### 1. 회원 관리 시스템 ✅
- **재학생 / 고용주 구분 인증**
  - 재학생: 학번, 학과, 학년 정보 입력
  - 고용주: 사업자명, 사업자번호 입력
- 회원가입, 로그인, 아이디/비밀번호 찾기
- 자동 로그인 (SharedPreferences)

**구현 화면**:
- `UserTypeSelectionActivity`: 사용자 유형 선택
- `SignupStudentActivity`: 재학생 회원가입
- `SignupEmployerActivity`: 고용주 회원가입
- `LoginActivity`: 로그인
- `FindIdActivity`, `FindPasswordActivity`: 계정 찾기

## 🔧 기술적 도전과 해결

### 도전 1: 일정 겹침 체크 알고리즘
**문제**: 요일과 시간을 기반으로 일정이 겹치는지 판단
**해결**: `Schedule` 모델에 `isConflictWith()` 메서드 구현
```java
public boolean isConflictWith(Schedule other) {
    if (this.dayOfWeek != other.dayOfWeek) return false;
    return !(this.endTime.compareTo(other.startTime) <= 0 ||
             this.startTime.compareTo(other.endTime) >= 0);
}
```

### 도전 2: 웹 크롤링 (학교 공지사항)
**문제**: 동양미래대학교 공지사항 자동 수집
**해결**: Jsoup 라이브러리 사용하여 HTML 파싱
```java
Document doc = Jsoup.connect(url).get();
Elements notices = doc.select(".notice-list .item");
```

### 도전 3: 사용자 유형별 UI 차별화
**문제**: 재학생과 고용주의 다른 메뉴 구성
**해결**:
- Bottom Navigation에서 동적으로 시간표 탭 추가/제거
- Fragment에서 사용자 유형에 따라 UI 변경
